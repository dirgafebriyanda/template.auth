# Template Auth

A modern authentication UI template built with HTML and Tailwind CSS.

## Preview

Live Demo: https://template-auth-ruddy.vercel.app/

## Features

* Login form
* Register form
* Smooth sliding animation
* Responsive design
* Tailwind CSS
* Clean and modern UI
* Easy customization

## Getting Started

### Clone Repository

```bash
git clone https://github.com/dirgafebriyanda/template.auth.git
cd template.auth
npm install
```

### Open Project

Open `index.html` directly in your browser.

Or run a local server:

```bash
npx serve
```

## Customization

You can easily customize:

* Colors
* Fonts
* Form fields
* Animations
* Layout
